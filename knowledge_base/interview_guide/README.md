# 面试宝典知识库

## 文件格式说明
- 支持 `.txt`, `.md` 文件
- 每个文件代表一个知识主题（如"技术面试技巧"、"行为面试STAR法则"等）
- 系统启动时会自动读取并分块向量化存储到 ChromaDB

## 示例文件结构

```
interview_guide/
├── 01_技术面试技巧.md
├── 02_STAR法则.md
├── 03_常见技术面试题.md
├── 04_行为面试指南.md
├── 05_薪资谈判技巧.md
└── 06_面试礼仪与着装.md
```

## 数据来源建议
- 《程序员面试宝典》
- 牛客网面试经验
- LeetCode 面试题解
- 各大公司面经汇总
